# Todo List 練習

[Demo 連結](https://onlystp417.github.io/todolist/)
## 說明

實作三個不同技術的 Todo list，藉以練習原生 JS，並透過 jQuery 與 Vue 實作，了解函式庫與框架的便利性，並比較 jQuery 與 Vue 事件驅動 V.S. 資料驅動之差異。

## 使用技術
- SCSS
- Vanilla JS
- jQuery
- Vue 3
- Vite 開發環境設置
